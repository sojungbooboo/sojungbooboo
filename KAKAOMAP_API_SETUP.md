# 카카오맵 API 설정 가이드

## 카카오맵 API Key 발급 방법

1. **카카오 개발자 콘솔 접속**
   - https://developers.kakao.com 접속
   - 카카오 계정으로 로그인

2. **내 애플리케이션 만들기**
   - "내 애플리케이션" → "애플리케이션 추가하기"
   - 앱 이름 입력 (예: "결혼식 청첩장")
   - 저장

3. **JavaScript 키 발급**
   - 생성한 앱 선택
   - "앱 키" 메뉴에서 "JavaScript 키" 복사

4. **플랫폼 등록**
   - "플랫폼" 메뉴 → "Web 플랫폼 등록"
   - 사이트 도메인 입력 (예: `https://sojungbooboo.github.io`)

5. **API 키 설정**
   - 발급받은 JavaScript 키를 `src/const.ts`의 `KAKAO_MAP_API_KEY`에 입력

## 과금 정보

- **무료 할당량**: 월 300,000건 (일반적인 웹사이트 사용에는 충분)
- **과금**: 무료 할당량 초과 시 건당 0.1원
- **결혼식 청첩장**: 월 300,000건은 충분히 사용 가능 (과금 걱정 없음)

## 사용 방법

API 키를 발급받으시면 `src/component/location/index.tsx` 파일을 수정하여 실제 지도를 표시할 수 있습니다.
